# Card

The base surface card — brand surface tint, hairline border, 14px radius. Compose freely with `Heading`, `Text`, `Tag`, `Button`. Set `bloom` on clickable cards for the brand hover motion (Breath Bloom scale + Iridescent Edge); `flush` removes padding for image-led layouts.

```tsx
<Card bloom>
  <Eyebrow accent>Offering</Eyebrow>
  <Heading size="md">Frequency calibration</Heading>
  <Text size="sm" muted>Daily practice, not theory.</Text>
</Card>
```
