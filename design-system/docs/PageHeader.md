# PageHeader

The editorial page opener: eyebrow with leading rule, `xl` display title in the brand headline font, muted lede, hairline bottom border. Children (buttons, a lockup) slot below the tagline.

```tsx
<PageHeader
  eyebrow="Brand Universe"
  title={<>Clarissa <em>Pierro</em></>}
  tagline="Five distinct worlds, one source — a consciousness guide, founder, and creator."
>
  <Button>Enter</Button>
</PageHeader>
```
