# Text

Body copy in Switzer — the one body font every brand shares (it's what keeps six very different headline fonts feeling like one company). `size="lede"` is the larger editorial intro paragraph; `muted` drops to secondary ink.

```tsx
<Text size="lede" muted>
  Five distinct worlds, one source.
</Text>
<Text size="sm">Free for commercial use.</Text>
```
