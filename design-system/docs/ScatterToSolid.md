# ScatterToSolid

Accent mark assigned to **Cotton Candy Revolt**: scattered gunpowder-and-forest dots breathing toward a solid Electric Orchid core. Designed as a hover/active state on cards and CTAs rather than a static logo — "scattered becomes whole."

```tsx
<ScatterToSolid size={48} />
```

Static under `prefers-reduced-motion`.
