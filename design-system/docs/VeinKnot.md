# VeinKnot

Accent mark assigned to **Web Development & Design**: wing-vein lines draw toward a meeting point, then an Electric Orchid bloom. Use for loading states, section dividers, and technical contexts — alongside the primary `EyeMark`, never instead of it.

```tsx
<VeinKnot size={56} />
```

For an actual loading indicator with a status label, prefer `VeinLoader` (it composes this mark). Animation loops (4.4s draw + bloom); static under `prefers-reduced-motion`.
