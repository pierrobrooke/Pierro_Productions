# Tag

Small label chip. `neutral` (default) is the quiet metadata chip from the Brand Universe cards ("Goddess Quiz", "Said To Her: Vol. I"); `accent` is the tiny uppercase Electric Orchid pill for statuses ("In development"); `outline` is hairline-bordered.

```tsx
<Tag>Wealth Currency Game</Tag>
<Tag variant="accent">In development</Tag>
<Tag variant="outline">Workshops</Tag>
```
