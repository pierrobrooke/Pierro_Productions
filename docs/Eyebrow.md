# Eyebrow

The tracked-uppercase kicker — the brand's signature "tracked Switzer" treatment. Sits above a `Display` title or labels a metadata line. `rule` adds the short leading dash; `accent` colors it with the brand accent.

```tsx
<Eyebrow rule>Brand Universe</Eyebrow>
<Eyebrow accent>New</Eyebrow>
```
