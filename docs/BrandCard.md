# BrandCard

The numbered portfolio row from the Brand Universe page: index number, name in the headline font with an optional status pill, description, metadata tags, hover arrow. Use for listing sub-brands, projects, or offerings — stack them in a bordered list.

```tsx
<BrandCard
  number="02"
  name="Zensual Mastery"
  pill="Separate project"
  description="The embodiment arm — where frequency becomes a daily, physical, undeniable practice."
  tags={["Proprietary methodology", "Goddess Quiz"]}
  href="/zensual"
/>
```
